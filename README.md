# Student-Information-Tracker

## Overview
The **Academic Records System** is a Python-based command-line application designed to manage student information efficiently. It allows users to add, view, and search student records, calculate average marks, and identify top-performing students. This project demonstrates fundamental programming concepts, data structures, and problem-solving skills in Python.

## Features
- **Add Student Records**: Store student ID, name, and marks.
- **View All Students**: Display all student information in a structured format.
- **Search Student by ID**: Quickly find a specific student using their unique ID.
- **Calculate Average Marks**: Automatically compute the class average.
- **Identify Top Performer**: Find the student with the highest marks.

## Technologies Used
- **Programming Language**: Python 3.8.8
- **Data Structures**: Dictionaries and Lists
- **Concepts Covered**: Loops, Functions, Conditional Statements, Input Handling
